# 📚 Smart Library Bot

A sophisticated AI-powered chatbot for searching and discovering books, magazines, and authors in a library system. Built with Python, FastAPI, and Natural Language Processing.

**Author:** Felipe Ravani  
**License:** MIT

---

## ✨ Features

- 🤖 **Intelligent Chatbot** - Natural language understanding for library queries
- 📖 **Book Search** - Find books by title, author, genre, or ISBN
- 📰 **Magazine Lookup** - Search magazines by topic, genre, or issue
- 👤 **Author Information** - Discover authors and their works
- 🔍 **Advanced Search** - Global search across all library items
- 💾 **Conversation History** - Track user interactions
- 🌐 **Web Interface** - Beautiful, responsive chat UI
- 📊 **REST API** - Fully documented API with Swagger/OpenAPI
- 🐳 **Docker Ready** - Easy deployment with Docker Compose
- ✅ **Unit Tests** - Comprehensive test suite

---

## 🛠️ Tech Stack

- **Backend:** FastAPI, Python 3.9+
- **Database:** SQLAlchemy ORM (SQLite by default)
- **NLP:** spaCy, NLTK
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **API Docs:** Swagger UI, ReDoc
- **Containerization:** Docker, Docker Compose
- **Testing:** pytest, pytest-asyncio

---

## 📋 Prerequisites

- Python 3.9 or higher
- pip (Python package manager)
- Docker & Docker Compose (optional, for containerized deployment)
- Git
